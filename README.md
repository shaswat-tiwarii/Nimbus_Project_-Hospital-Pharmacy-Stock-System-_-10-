**Project Overview**

This project is a Pharmacy Inventory Management System written in C.
It allows a pharmacy to manage its stock of medicines, track expiry dates, update quantities, and store data in a binary file for future use.

**Features**

- Add a new medicine record
- Edit existing product details (name, price, expiry)
- Display all inventory items with expiry status
- Show only expired medicines
- Automatically delete expired items
- Restock items (increase quantity)
- Purchase item (decrease stock & display bill)
- Save inventory data to file
- Load saved data at any time
- Demo data loading for quick testing

**How to Run**
*Compile the program*

Using GCC:
gcc store.c -o store

*Run the program*

./store
